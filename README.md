# StudentManagementSystem
The Student Management System is a desktop application designed to manage student records efficiently. It is built using Java Swing for the graphical user interface (GUI) and MySQL for database management. The application provides a seamless experience for users to:
•	Sign up and log in using their credentials.
•	Register student details such as name, course, gender, and hobbies.
•	View and manage registered student information stored in a MySQL database.
The system is designed to be user-friendly, with a focus on simplicity and functionality. It ensures data persistence by storing all information in a relational database, making it easy to retrieve and update records as needed.
